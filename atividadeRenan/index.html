Linguagem de marcação HTML

HTML é a sigla para HyperText Markup Language, que significa Linguagem de Marcação de Hipertexto.É uma linguagem de programação que define a estrutura e o significado do conteúdo de uma página web.

O HTML é o componente básico da web, permitindo a criação de websites e a inserção de novos conteúdos, como imagens e vídeos.

Vantagens do HTML

  • Fornece uma estrutura sólida para o conteúdo web, permitindo a organização e hierarquização do conteúdo
  • Acessibilidade
  • O HTML é compatível com todos os navegadores e dispositivos.

Desvantagens do HTML

  1. Não trabalha com operações lógicas
  2. Para implementar funções dinâmicas, precisa ser aliado a outras linguagens de back end
  3. Alguns navegadores mais antigos não renderizam novas tags
